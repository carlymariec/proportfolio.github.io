# 📁 How to Upload Your Files

Upload your files directly to this repository using the GitHub web interface
(click **Add file → Upload files** on the folder page).  
Once a file is in the right place with the right name, the website will link to it automatically — no code changes needed.

---

## 🖼️ Profile Photo

| Upload to | File name |
|-----------|-----------|
| `images/` | `profile.jpg` (or `profile.png`) |

**Example path:** `images/profile.jpg`

> Recommended: square crop, at least 400 × 400 px.  
> Both `.jpg` and `.png` are supported.

---

## 📄 Resume

| Upload to | File name |
|-----------|-----------|
| `files/resume/` | `resume.pdf` |

**Example path:** `files/resume/resume.pdf`

The **"Download Resume"** button on the website will link directly to this file.

---

## 📂 Assignment Samples

Upload each assignment as a PDF inside the `files/assignments/` folder.  
Use the exact file names shown below so the website links them correctly:

| Assignment card on website | Upload to | File name |
|---------------------------|-----------|-----------|
| Legal Memorandum – Case Law Analysis | `files/assignments/` | `legal-memo.pdf` |
| Case Brief – Civil Litigation | `files/assignments/` | `case-brief.pdf` |
| Statutory Research Report | `files/assignments/` | `statutory-research.pdf` |
| Criminal Procedure Documentation | `files/assignments/` | `criminal-procedure.pdf` |

**Example path:** `files/assignments/legal-memo.pdf`

> All documents should be sanitized and redacted before uploading, as noted in the README.
